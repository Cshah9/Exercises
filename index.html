Step 3: Branch the master branch

Create a branch of the master branch with the following command. For this example, I am using "ghostwriter" for the new user and branch. You should substitue it with your GitHub user name.

git branch [your username]-branch

Step 4: Checkout your new branch

Now that you have created a new branch, you have to switch from the master branch to the new branch in your LOCAL repository so that the changes you commit will be commited to the branch. Use the git checkout command for that.

git checkout [your username]-branch
